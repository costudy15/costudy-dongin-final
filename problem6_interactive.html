문제
최고차항의 계수가 $1$인 사차함수 $f(x)$와 이차함수 $g(x)=2x^2-x-4$가 다음을 만족한다.
(가) 곡선 $y=f(x)-g(x)$는 $x$좌표가 $2$인 점에서 $x$축에 접한다.
(나) 함수 $y=|f(x)-g(x)|$는 실수 전체의 집합에서 미분가능하다.
$f'(0)=2$일 때 $f(1)$의 최댓값은 $\alpha$이다. $40\alpha$의 값을 구하시오.
1단계(가) — x=2에서 접 → (x−2)² 인수 핵심조건 활용
💭 먼저 생각해보세요$y=f(x)-g(x)$가 $x=2$에서 $x$축에 접한다 — 접한다는 건 그 점에서 함숫값과 미분계수가 모두 0이라는 뜻이죠. 인수로는?
이 단계로 넘어가는 이유 — 접점이므로 $h(2)=0$이고 $h'(2)=0$ → $x=2$가 중근 → $(x-2)^2$을 인수로 갖는다. $h=f-g$는 사차(최고차항 1)이므로 나머지는 이차식.
$$h(x)=f(x)-g(x)=(x-2)^2(x^2+ax+b)$$
접선과 중근인수분해
2단계(나) — |h| 미분가능 → h≥0 → 판별식 ≤0 핵심조건 활용
💭 먼저 생각해보세요$|h|$가 모든 점에서 미분가능하려면, $h$의 그래프가 $x$축을 어떻게 지나야 할까요? (뾰족점이 생기면 안 돼요)
이 단계로 넘어가는 이유 — $|h|$가 꺾이는 건 $h$가 부호를 바꾸며 $x$축을 가로지를 때. 미분가능하려면 부호를 안 바꿔야 하므로 $h\ge0$. $(x-2)^2\ge0$이니 $x^2+ax+b\ge0$ → 판별식 조건.
$$x^2+ax+b\ge0\ (\forall x)\ \Rightarrow\ D=a^2-4b\le0\quad\cdots\ \text{㉠}$$
절댓값 함수의 미분가능이차식의 부호와 판별식
3단계f′(0)=2 조건 대입계산
💭 먼저 생각해보세요$f=h+g$이고 $g'(x)=4x-1$. $f'(0)$을 $a,\ b$로 나타내 조건에 넣으면?
$f'(x)=2(x-2)(x^2+ax+b)+(x-2)^2(2x+a)+g'(x)$, $x=0$ 대입 → $f'(0)=-4b+4a-1$.
$$-4b+4a-1=2\ \Rightarrow\ b=a-\tfrac34\quad\cdots\ \text{㉡}$$
곱의 미분도함수 값
4단계㉠·㉡ 결합 → a의 범위계산
💭 먼저 생각해보세요㉡의 $b$를 ㉠에 대입하면 $a$만의 부등식이 돼요.
$$a^2-4\!\left(a-\tfrac34\right)=a^2-4a+3\le0$$
$$(a-1)(a-3)\le0\ \Rightarrow\ 1\le a\le3$$
이차부등식
5단계f(1)의 최댓값목표 역산
💭 먼저 생각해보세요$f(1)$을 $a,\ b$로 쓰고 ㉡로 $b$를 없애면 $a$의 일차식. $1\le a\le3$에서 최대는?
이 단계로 넘어가는 이유 — 구하는 건 $f(1)$의 최댓값 → $f(1)$을 $a$ 하나의 식으로 환원해 범위에서 최댓값을 찾는다.
$$f(1)=(1-2)^2(1+a+b)+g(1)=a+b-2=2a-\tfrac{11}{4}$$
$$a=3\ \text{일 때 최대}\ \tfrac{13}{4}=\alpha\ \Rightarrow\ 40\alpha=130$$
일차식의 최대대입
정답   40α = 130  (α = 13/4)
