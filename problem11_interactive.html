11. 빈칸 채우기 — 증가 조건·2실근으로 F(5)+G(4)+k · COSTUDY 수II · 삼차함수의 증가와 실근
탭하면 다음 ▸
문제
두 실수 $a\,(a\ne0),\ b$에 대하여 $f(x)=\tfrac13 ax^3-bx^2-(a-4)x-3a^2,\ \ g(x)=(a+b)x^2-2(a+2)x$가 다음을 만족한다.
$f(x)$는 실수 전체에서 증가하고, $f(x)+g(x)=0$이 서로 다른 $2$개의 실근을 갖는다.
빈칸 $(가)=F(a),\ (나)=G(a),\ (다)=k$일 때 $F(5)+G(4)+k$의 값을 구하시오.
1단계(가) f가 증가 → 판별식 ≤ 0 핵심조건 활용
💭 먼저 생각해보세요$f$가 실수 전체에서 증가하려면 $f'(x)\ge0$이 항상 성립해야 해요. $f'(x)=ax^2-2bx-(a-4)$가 항상 0 이상이려면?
이 단계로 넘어가는 이유 — 이차식 $f'$이 항상 $\ge0$ ⇔ $a\gt0$이고 판별식 $D\le0$. 이 $D$의 빈칸이 $(가)$.
$$D=(-2b)^2-4a\{-(a-4)\}=4b^2+\underbrace{(4a^2-16a)}_{(\text{가})}\le0$$
삼차함수의 증가이차식의 부호·판별식
2단계(나) f+g=0 정리계산
💭 먼저 생각해보세요$f(x)+g(x)$를 정리하면 삼차식이 돼요. $a\gt0$로 양변을 나눠 깔끔히 만들면 우변에 무엇이 남죠?
$f+g=\tfrac13 ax^3+ax^2-3ax-3a^2=0$, $\times\tfrac{3}{a}$ → $x^3+3x^2-9x-9a=0$.
$$x^3+3x^2-9x=\underbrace{9a}_{(\text{나})}$$
식 정리삼차방정식
3단계h(x)=x³+3x²−9x 의 증감계산
💭 먼저 생각해보세요$h(x)=x^3+3x^2-9x$의 도함수로 극대·극소를 찾으면?
$$h'(x)=3(x+3)(x-1)=0\ \Rightarrow\ x=-3,\ 1$$
$$\text{극대}\ h(-3)=27,\qquad \text{극소}\ h(1)=-5$$
삼차함수의 극값
4단계(다) 서로 다른 2실근 → a=3 핵심목표 역산
💭 먼저 생각해보세요$x^3+3x^2-9x=9a$가 서로 다른 2실근 — 수평선 $y=9a$가 $h$와 정확히 2점에서 만나려면 어디를 지나야 할까요?
이 단계로 넘어가는 이유 — 삼차곡선과 수평선이 서로 다른 2점에서 만나는 건 극값을 지날 때(접점+교점)뿐. 그래서 $9a=27$ 또는 $9a=-5$. $a\gt0$이므로 $a=3$ → $(다)\ k=3$.
$$9a=27\ \text{또는}\ 9a=-5,\quad a\gt0\ \Rightarrow\ a=3$$
실근의 개수와 극값그래프와 직선
5단계b 범위와 F(5)+G(4)+k계산
💭 먼저 생각해보세요$a=3$을 $\text{㉠}$에 넣어 $b$ 범위를 구하고, 마지막에 $F(5)+G(4)+k$를 계산해요.
$4b^2+4(9)-16(3)=4b^2-12\le0 \Rightarrow b^2\le3 \Rightarrow -\sqrt3\le b\le\sqrt3$.
$$F(a)=4a^2-16a,\quad G(a)=9a,\quad k=3$$
$$F(5)+G(4)+k=20+36+3=59$$
대입최종 계산
정답   F(5)+G(4)+k = 59
